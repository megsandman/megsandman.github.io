Every object has a class, which is a data structure that groups together objects that have the same behaviors and properties. Ruby allows users to define their own classes but also has several built in classes such as strings, integers, arrays, and hashes.

Learning and thinking styles.
August 24, 2014

Before starting Phase 0, Dev Bootcamp asked students to read up on the Gregorc Model and take a quiz to determine our own thinking style. The Gregorc Model looks at thinking styles along two dimensions: perceptual quality, which can be either concrete or abstract, and ordering ability, which can be either sequential or random. My personal thinking style is concrete sequential. What does that mean? I like order. I like structure. I don't do well with ambiguity or abstract ideas.
