Reflection: I really enjoyed this challenge and it gave me a much better sense of some of the challenges web designers face now that there is seemingly never ending number of devices from which users can view web sites. The exercise helped solidify my understanding of design principles for the different screen sizes and how to differentiate between these sizes in the CSS file. We initially ran into difficulty when adjusting the format for the mobile screen in getting the nav section above the content. Positioning of elements on the screen is an area where I still have a lot to learn so this exercise proved to be great practice in that area. By removing the left float on the nav section and changing it to a block we were able to get the section to move from the left of the content section to above the section.
